# DECO1400-Car-Assistant

## Project Overview
The **Car Assistant Platform** is a specialized web application designed to help users navigate the complex decision-making process of choosing between ICE (Internal Combustion Engine), EV (Electric Vehicle), and Hybrid cars. The platform focuses on budget alignment, usage habits, and long-term maintenance cost transparency.

## Technical Implementation (Week 8 Status)
- **Semantic HTML5**: Using `<nav>`, `<main>`, and `<section>` to ensure a logical document structure and high accessibility.
- **CSS Grid & Flexbox**: Implementing a macro-layout with CSS Grid for the persistent sidebar and main content area. Flexbox is used for micro-layout alignment within components.
- **Responsive Design Plan**: Utilizing CSS Media Queries to ensure the site is functional across desktop, tablet, and mobile devices.
- **Vanilla JavaScript**: Planning for 4 unique interactions, including a "Match Score" calculator and dynamic price filters.

## Design Principles
- **Visibility of System Status**: A fixed sidebar ensures users always know their current location within the site.
- **Match Between System and Real World**: Using non-technical terminology (e.g., "Budget", "Daily Use") to accommodate average car buyers.
- **Aesthetic and Minimalist Design**: Leveraging a clean UI to reduce cognitive load during complex comparisons.
